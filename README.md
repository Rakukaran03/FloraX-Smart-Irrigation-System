🌱 FloraX – Smart Irrigation System

🚀 Overview
FloraX is a smart irrigation system designed to automate watering using zone-based soil moisture monitoring. It supports both standalone operation and cloud-based monitoring.

🔥 Key Features
- Zone-Based Automatic Watering
- Self-Powered Smart Valves (Solar)
- Works Without Internet (Offline Mode)
- ESP32-Based Distributed Control
- Optional Cloud Monitoring & Mobile App
- Water Usage Tracking (Flow Sensor)

🏗️ System Architecture
Branches (ESP32 C3):
- Soil moisture sensing
- Local valve control

Brain (ESP32 WROOM):
- Data aggregation
- Cloud communication
- User interface

🧠 Versions
- **Standard**: Autonomous branches only
- **Smart**: Branches + Cloud monitoring
- **Pro**: Brain + Branches (Full system)

🛠️ Tech Stack
- ESP32 (C3 + WROOM)
- WiFi Communication
- Soil Moisture Sensors
- Servo Motor Valves
- Water Flow Sensor
- Web Dashboard
- Humidity Sensor

🎯 Applications
- Home Gardens
- Smart Agriculture
- Greenhouses
- Water Conservation Systems

📈 Future Improvements
- AI-based watering prediction
- Weather integration
- Mobile App Expansion
